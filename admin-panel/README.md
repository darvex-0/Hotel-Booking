# 🛠️ Beach Resort - Management Dashboard

The command center for Beach Resort operations. This React-based dashboard allows administrators to manage rooms, bookings, users, and guest reviews with ease.

## ✨ Features

- **Real-time Analytics**: Monitor bookings and revenue at a glance.
- **Room Management**: Add, update, or remove rooms and manage availability.
- **Booking Control**: Review guest reservations and update order statuses.
- **User Management**: Manage guest accounts and administrative roles.
- **Responsive Layout**: Fully functional on desktop and tablet for on-the-go management.

## 🛠️ Technology Stack

- **Framework**: React.js (CRA)
- **UI Library**: Ant Design
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Notifications**: React Toastify

## ⚙️ Setup & Installation

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Configuration**:
   Create a `.env` file with the following:
   ```env
   REACT_APP_API_URL=http://localhost:5000/api/v1
   ```

3. **Run the Dashboard**:
   ```bash
   npm start
   ```
